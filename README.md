# Hostel Management System

The **Hostel Management System** is a full-stack project built using **Spring Boot** and **React**.  
It digitizes and streamlines the management of hostels, buildings, floors, rooms, and beds with clean APIs and a user-friendly frontend.

---

## Features

### **Hostel Module**
- Add, view, and manage hostels  
- View details of a selected hostel  
- List all buildings inside the hostel  
- Add new buildings for any hostel  

### **Building Module**
- View building details  
- List floors inside the building  
- Add new floors  

### **Floor Module**
- View floor details  
- Manage number of rooms  
- Add rooms inside floors  

### **Room Module**
- Manage room details  
- Add and allocate beds  

### **Bed Module**
- Manage bed allocation  
- Track bed status / availability  

---

## Tech Stack

### **Backend**
- Spring Boot  
- Spring JPA / Hibernate  
- MySQL / H2 (as DB)  
- REST APIs
